【C/CPP】友元函数和友元类
Fan Engineer
  2023-01-09 23:18:34 2023-01-09 23:18   2023-01-10 22:44:35  
  • cpp 
 

友元（friend）可以使其他类中的成员函数以及全局范围内的函数访问当前类的 private 成员。

友元函数

友元函数可以是不属于任何类的非成员函数，也可以是其他类的成员函数，在类中声明时要在前面加 friend 关键字。

将非成员函数声明为友元函数

show() 是一个全局范围函的非成员函数，不属于任何类，声明为Student类的友元函数后即可访问类中的private成员。

注意，友元函数不同于类的成员函数，在友元函数中不能直接访问类的成员，必须要借助对象。

成员函数调用时会隐式的增加 this 指针，才能访问到该对象的成员；而 show() 是非成员函数，没有 this 指针，所以无法访问成员，必须通过参数传递对象来访问。

将其他类的成员函数声明为友元函数

在 Student 类中使用到了 Address 类，所以要提前进行声明。Student 类中的成员函数 show() 使用了 Address 的成员，所以在定义之前要先声明 Address 类。

类只有在正式声明后才能创建对象，在对一个类作了提前声明后，可以用该类的名字去定义指向该类型对象的指针变量或引用变量。

一个函数可以被对各类声明为友元函数，即可以访问多个类中的 private 成员。

友元类

将整个类声明为另一个类的朋友，就是友元类。友元类中的所有成员函数都是另一个类的友元函数。

需要注意：

  • 友元的关系是单向的而不是双向的。
  • 友元的关系不能传递。

一般不建议把整个类声明为友元类，而只将某些成员函数声明为友元函数。
