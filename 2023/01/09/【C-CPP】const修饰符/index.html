如果不希望某些数据被修改，可以使用const关键字限定。

const 成员变量和成员函数

const成员变量在初始化时只能通过构造函数的初始化列表。

const成员函数可以使用类中的所有成员变量，但是不能修改其值，主要是为了保护数据设置。通常将get函数设置成常成员函数。

常成员函数在声明和定义的时候需要在函数头部后面加上const关键字。

注意 const 的位置区分：

  • 函数开头的 const 修饰的是函数的返回值，表示返回值是const类型，不能被修改，例如const char* getName()。
  • 函数头部结尾的 const 表示常成员函数，只能读取成员变量的值，不能修改成员变量。

const 对象

const修饰对象称为常对象，对象被定义为常对象只能调用类的 const 成员（包括 const 成员变量和 const 成员函数）。

无论哪种形式，只要定义为常对象就只能访问被 const 修饰的成员。
