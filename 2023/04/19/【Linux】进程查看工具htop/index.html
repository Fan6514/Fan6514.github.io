【Linux】进程查看工具htop

htop 是一个类似于 Linux top 命令的交互式进程查看工具，它可以显示系统中所有进程的信息，并且支持通过鼠标或键盘来进行交互式操作。

在使用 htop 命令时，可以通过命令行选项来指定需要显示的进程信息和显示方式，也可以通过交互式操作来进行实时的进程监控和管理。

安装

  1. Ubuntu
  1. Centos

如何在CentOS 7上安装Htop (vue5.com)

使用

htop 常用的一些命令行选项：

  • -u, –user=USERNAME：只显示指定用户的进程信息；
  • -p, –pid=PID：只显示指定进程 ID 的信息；
  • -d, –delay=DELAY：设置更新延迟时间，默认为 2 秒；
  • -C, –no-color：禁用颜色显示；
  • -h, –help：显示帮助信息。

htop 常用的一些交互式操作：

  • 使用箭头键或 PgUp/PgDn 来滚动进程列表；
  • 使用 F1 键来查看帮助信息；
  • 使用 F2 键来进入设置界面，可以进行显示和操作方式的配置；
  • 使用 F3 键来进行进程搜索，输入关键字后按 Enter 键；
  • 使用 F4 键来进行进程筛选，可以按用户、进程状态等条件进行筛选；
  • 使用 F5 键来刷新进程列表；
  • 使用 F9 键来发送信号给选中的进程；
  • 使用 F10 键或 q 键来退出 htop。
